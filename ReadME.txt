Overview

This project involves scraping historical Olympics data from Google, cleaning the dataset, and creating a report using Power BI. 
The goal was to organize the data and present it in an informative and visually appealing way.

Files Included

Python codes: Script used to scrape data from Google.
Data Files: These are the raw and processed data files used in the project.
Power BI Report: The Power BI file where the data is visualized and analyzed.

Project Steps

Uploading Data: Uploaded csv file to powerBI 
Data Cleaning: Fixed and cleaned the data columns to ensure accuracy and consistency.
Power BI Report: Created a detailed report to visualize and analyze the cleaned data.
